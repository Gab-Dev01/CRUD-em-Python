# CRUD em Python

Este é um simples sistema de CRUD (Criar, Ler, Atualizar, Deletar) implementado em Python, utilizado para gerenciar uma lista de usuários. O programa permite ao usuário cadastrar novos nomes, listar os cadastrados, editar ou excluir registros, tudo através de um menu interativo no terminal.

## Funcionalidades

- **Cadastrar:** Adiciona um novo usuário à lista.
- **Listar:** Exibe todos os usuários cadastrados.
- **Editar:** Permite alterar o nome de um usuário já cadastrado.
- **Excluir:** Remove um usuário da lista.
- **Sair:** Encerra o programa.

## Como Usar

1. Execute o arquivo no VsCode ou em um sistema de sua preferência.
